ลากกรอบบนรูปภาพเพื่อสร้างเมนูมากกว่า 6 ช่อง (สูงสุด 20 ช่อง)

วิธีใช้: กดคลิกเมาส์ค้างแล้วลากเพื่อสร้างกรอบปุ่มกด

2. ตั้งค่าข้อความแต่ละช่อง

ยังไม่มีการวาดกรอบเมนู ลากเมาส์บนรูปเพื่อเพิ่มช่อง

3. โค้ดผลลัพธ์ (JSON)

วิธีนำโค้ด JSON ไปใช้งานผ่าน LINE Messaging API

การทำ Rich Menu แบบมากกว่า 6 ช่อง ต้องสั่งการผ่าน API ของ LINE นี่คือขั้นตอนแบบสรุปครับ:

ขั้นตอนที่ 1: สมัคร LINE Developers

  1. ไปที่ LINE Developers Console
  2. ล็อกอินด้วยบัญชี LINE ของคุณ สร้าง Provider และสร้าง Channel ประเภท Messaging API
  3. ไปที่แท็บ Messaging API เลื่อนลงมาล่างสุด กดสร้าง Channel access token (long-lived) และก๊อปปี้เก็บไว้

ขั้นตอนที่ 2: ใช้โปรแกรม Postman เพื่อส่ง API (สร้างโครงสร้างเมนู)

  1. เปิด Postman หรือแอปจำลองการส่ง API
  2. ตั้งค่า Request:
    • Method: POST
    • URL: https://api.line.me/v2/bot/richmenu
  3. ไปที่แท็บ Headers เพิ่ม:
    • Key: Authorization / Value: Bearer {ใส่ Channel access token ของคุณ}
    • Key: Content-Type / Value: application/json
  4. ไปที่แท็บ Body เลือก raw และวางโค้ด JSON ที่คัดลอกมาจากเครื่องมือด้านซ้าย
  5. กดปุ่ม Send (ถ้าสำเร็จ ระบบจะตอบกลับมาเป็น richMenuId ให้ก๊อปปี้เก็บไว้)

ขั้นตอนที่ 3: อัปโหลดรูปภาพใส่ Rich Menu

  1. สร้าง Request ใหม่ใน Postman:
    • Method: POST
    • URL: https://api-data.line.me/v2/bot/richmenu/{ใส่ richMenuId ที่ได้จากข้อ 2}/content
  2. แท็บ Headers:
    • Key: Authorization / Value: Bearer {Channel access token}
    • Key: Content-Type / Value: image/jpeg (หรือ image/png)
  3. แท็บ Body เลือก binary แล้วอัปโหลดไฟล์รูปภาพของคุณ
  4. กด Send (ถ้าสำเร็จจะขึ้น status 200 OK)

ขั้นตอนที่ 4: สั่งเปิดใช้งาน (Set Default)

  1. สร้าง Request ใหม่ใน Postman:
    • Method: POST
    • URL: https://api.line.me/v2/bot/user/all/richmenu/{ใส่ richMenuId}
  2. แท็บ Headers: ใส่แค่ Authorization แบบเดียวกับข้อบนๆ
  3. กด Send เสร็จสิ้น! เมนูจะไปแสดงที่หน้าแชทของทุกคน
